    Project Info¶

    The devbox info command provides detailed information about a project, including its sources and mounted volumes.

    Usage¶

    # Get info for current project (auto-detected)
devbox info

# Get info for specific project
devbox --name project-name info

    Output Sections¶

    Sources¶

    Shows information about all source repositories in the project:

     Sources:
┌────────────────┬────────────────────────┬──────────────┬─────────────┐
│ Name           │ Message                │ Author       │ Date        │
├────────────────┼────────────────────────┼──────────────┼─────────────┤
│ api-service    │ Update API endpoints   │ John Doe     │ 2 days ago  │
│ worker (jobs)  │ Add job processor     │ Jane Smith   │ 1 hour ago  │
└────────────────┴────────────────────────┴──────────────┴─────────────┘

    Mounts¶

    Displays active local mounts:

     Mounts:
┌────────────────────┬───────────────────────────────┐
│ Mount path         │ Local path                    │
├────────────────────┼───────────────────────────────┤
│ ./sources/api      │ /Users/dev/code/api-service   │
│ ./sources/worker   │ /Users/dev/code/worker        │
└────────────────────┴───────────────────────────────┘

    Features¶

    • Shows Git status for all source repositories
    • Displays active local mounts
    • Includes sparse checkout information
    • Updates project configuration before showing info

    Common Use Cases¶

    Check Source Status¶

    # View status of all sources
devbox info

    Verify Mounts¶

    # Check which sources are mounted locally
devbox info | grep "Mount"

    Project Health Check¶

    The command performs several checks before showing information: * Updates project configuration * Updates host file entries * Updates SSL certificates * Verifies source repositories

    Automatic Detection¶

    Like other DevBox commands, info supports automatic project detection:

    # DevBox will detect the project based on:
# - Current directory
# - Git remote URLs
# - Local mounts
cd /path/to/service
devbox info
